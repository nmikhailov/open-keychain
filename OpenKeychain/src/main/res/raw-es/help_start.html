Primeros pasos

Primero, necesita una clave personal. Cree una a través del menú en "Claves" o importe claves secretas (privadas) existentes. Después, puede descargar las claves de sus amigos o intercambiarlas mediante códigos QR o transmisión NFC.

Sobre un Android inferior a la 4.4, se recomienda que instale OI File Manager para una selección de ficheros mejorada. Para compartir mediante códigos QR instale Barcode Scanner. Haga clic en los enlaces para abrir la Google Play Store o F-Droid para instalarlos.
